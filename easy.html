HTML - easy

The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

VERSIONS:

  • HTML 2.0 - Published by the IETF as RFC 1866 with Proposed Standard status (November 24, 1995);
  • HTML 3.0 - March 28, 1995 - IETF Internet Draft (up to September 28, 1995)
  • HTML 3.2 - January 14, 1997;
  • HTML 4.0 - December 18, 1997;
  • HTML 4.01 - December 24, 1999;
  • ISO / IEC 15445: 2000 (so called ISO HTML, based on HTML 4.01 Strict) - May 15, 2000;
  • HTML5 - October 28, 2014;
  • HTML 5.1 began development on December 17, 2012. Recommended for use since November 1, 2016;
  • HTML 5.2 was introduced on December 14, 2017;
  • HTML 5.3 was released on December 24, 2018;

There is no official HTML 1.0 specification. Before 1995, there were many unofficial HTML standards. To make the standard version different from them, it was immediately assigned a second number.
